# Vite React Demo App

This is a minimal Vite + React + TypeScript demo application that displays listings based on the seed data provided in the properties.csv file.

## How to Run

1. Clone the repository.
2. Run `npm install`.
3. Start the application with `npm run dev`.

## Files Added
- properties.csv
- src/*
- vite.config.js
- README.md

## Seed Data
The seed data for properties can be found in the properties.csv file.
